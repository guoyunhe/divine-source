# divine-source

## Install

### Run directly

```bash
npx divine-source
```

### Install on system

```bash
npm i -g divine-source
divine-source
```

### Install in project

```bash
npm i -D divine-source
```

Add script entry:

```json
{
  "scripts": {
    "divine-source": "divine-source"
  }
}
```

Run:

```bash
npm run divine-source
```

## Options

### `--help`

Show help

### `--version`

Show version
